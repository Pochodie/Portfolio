## About
This repository contains a static portfolio website built with plain HTML, CSS, and JavaScript. It provides pages for a home/index, projects, gallery, music, and contact.

## Features
- Home / landing page
- Projects listing
- Image gallery
- Music page / player
- Contact page
- Static assets (images, fonts, etc.) and separate folders for CSS and JS

## Included Files
- `index.html` — Main page
- `projects.html` — Project showcase
- `gallery.html` — Image gallery
- `music.html` — Music / audio page
- `contact.html` — Contact form / contact details
- `css/` — Stylesheets
- `js/` — JavaScript files
- `assets/` — Images and other static assets
- `.gitignore`

## Built With
- HTML
- CSS
- JavaScript

## Getting Started
To preview the site locally you only need a web browser.

Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)

## Run Locally
1. Clone the repository:
   - git clone https://github.com/Pochodie/Portfolio.git
2. Open `index.html` in your browser:
   - Double-click `index.html` or open it via your editor/IDE.
